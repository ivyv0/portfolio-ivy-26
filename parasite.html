Living Above,
Surviving Below

An interactive website inspired by the film Parasite, exploring the contrast between wealth and poverty through scroll, space, typography and cinematic motion.

Visual Storytelling Typography Interactive Website
Type

Interactive Website

Focus

Visual Storytelling

Theme

Class Difference

Context

Living Above, Surviving Below is an interactive website about the film Parasite. The project focuses on the contrast between rich and poor, using the house as a visual symbol for society.

The website starts at the villa of the wealthy Park family. This villa represents luxury, calmness and control. As the user scrolls, the camera slowly zooms towards the house, making it feel as if the visitor is entering the world of the film.

Interaction

Scrolling is an important part of the experience. The further the user scrolls, the darker the website becomes. The journey moves from the villa, to the stairs, and finally down into the basement.

This downward movement reflects the theme of Parasite: some people live above in comfort, while others have to survive below. The staircase becomes a metaphor for class difference. Everyone wants to move up in life, but not everyone is given the same chances.

Visual Direction

I chose a dark and cinematic style to create tension. Large typography, layered visuals and subtle movement are used to make the visitor feel that something is hidden beneath the surface.

Concept

The house seems beautiful and perfect at first, but underneath it is a hidden basement where someone invisible lives. This contrast shows that behind wealth, there can also be hidden problems and inequality.

My goal was to help people understand the film more deeply, even if they have not seen Parasite yet. The website shows that the house is not just a location, but a symbol for society: wealth exists above, poverty exists below, and between them is a lot of hidden tension.

Final Message

“The system was never built for everyone to rise.”
